    - Corrected tests setup
- Upgraded imio.dms.mail 3.1.5
    - Added defer_security_update parameter on IMPreManagerValidation wf adaptation
    - Added "omail_signers_origin" option with 3 modes to define outgoing mail signers.
    - Removed useless settings descriptions
    - Refactored ContentCategoryGroup and ContentCategory creation
    - Fixed approver userid in @@manage-approvals.
    - Fixed start_approval_process audit log on unapprove_file.
    - Improved import_sign_examples to handle multiple signers
    - Do not display ownership fieldset
    - Removed IScanFields on annex type
    - Updated incoming mail_types and send_modes examples.
    - Fixed AttributeError (get_person) raised in is_hp_used_in_signer_rules (PARAF-477).
    - Allow removal of userid on a person if he's not in any group.
    - Fixed duplicate transitions on wfa reapply in migration 3.1.
- Upgraded imio.esign 1.0b10
    - Fixed session size computation when an annex is deleted.
    - In `@@session-annotation-info` display `UID` next to link to element.
    - Display annex description in `@@parapheo` quick look column.
    - Added red/green icon next to signers to show if not signed or signed.
- Upgraded imio.zamqp.dms 0.25.0
    - Added temp dir cleanup
    - Added test_outgoing_mail.
    - Used im send_modes field
- Upgraded plonetheme.imioapps 2.65
    - PARAF-476: added styles for `span.signer-not-signed`, `span.signer-signed` and `span.signer-refused`.
